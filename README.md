# ShadowView


#### 引用
```
compile 'com.dongze:shadowlib:1.0.0'
```

#### 支持版本
```
Android4.1以上
```

#### 使用方法：
```
View view = findViewById(R.id.shadow_layout);
ShadowView.with(this, view).initView();
```
